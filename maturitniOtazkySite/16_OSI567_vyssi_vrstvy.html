OSI vrstva 5 – Relační (Session Layer)

Funkce

  • Řízení dialogu – simplex, half-duplex, full-duplex komunikace
  • Správa relací – navázání, udržování a ukončení sezení
  • Synchronizace – checkpointy pro obnovení přenosu při přerušení
  • Token management – řízení, kdo má právo vysílat

Příklady protokolů a technologií

  • NetBIOS, RPC (Remote Procedure Call)
  • PPTP (Point-to-Point Tunneling Protocol)
  • SQL session management, SIP (Session Initiation Protocol)

Soketové programování

Socket = koncový bod komunikace identifikovaný jako (IP adresa, port, protokol).

TCP Server/Klient (Python)

# TCP Server
import socket

server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)  # IPv4, TCP
server.bind(('0.0.0.0', 12345))
server.listen(5)                     # max 5 čekajících spojení
print("Server naslouchá na portu 12345...")

conn, addr = server.accept()        # blokující volání – čeká na klienta
print(f"Připojen: {addr}")
data = conn.recv(1024)              # přijmi max 1024 bajtů
conn.send(b"Odpoved od serveru")
conn.close()
server.close()

# TCP Klient
import socket

client = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
client.connect(('127.0.0.1', 12345))
client.send(b"Ahoj servere!")
odpoved = client.recv(1024)
print(odpoved.decode())
client.close()

UDP Server/Klient (Python)

# UDP Server
import socket

server = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)   # UDP
server.bind(('0.0.0.0', 12345))
data, addr = server.recvfrom(1024)      # přijmi datagram
server.sendto(b"Odpoved", addr)         # odešli odpověď

# UDP Klient
client = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
client.sendto(b"Ahoj", ('127.0.0.1', 12345))
data, addr = client.recvfrom(1024)
print(data.decode())

Workflow socketu

TCP Server:  socket() → bind() → listen() → accept() → recv()/send() → close()
TCP Klient:  socket() → connect() → send()/recv() → close()

UDP Server:  socket() → bind() → recvfrom()/sendto() → close()
UDP Klient:  socket() → sendto()/recvfrom() → close()

Serializace dat

import json
data = {"jmeno": "Jan", "vek": 20}
json_str = json.dumps(data)             # → '{"jmeno": "Jan", "vek": 20}'
parsed = json.loads(json_str)           # → dict

# XML
import xml.etree.ElementTree as ET
root = ET.fromstring('<student><jmeno>Jan</jmeno></student>')
print(root.find('jmeno').text)          # → Jan

AAA model

  • Authentication – ověření identity (kdo jsi?)
  • Authorization – oprávnění (co smíš?)
  • Accounting – logování (co jsi dělal?)
V praxi (TCP/IP model): Vrstvy 5–7 jsou sloučeny do aplikační vrstvy. Většina moderních protokolů (HTTP, TLS) pokrývá funkce všech tří OSI vrstev.
