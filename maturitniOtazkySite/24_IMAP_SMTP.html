Cesta e-mailu

Odesílatel (MUA)                                         Příjemce (MUA)
     |                                                          ↑
     | SMTP (port 587)                              IMAP (port 993)
     ↓                                                          |
Mail Server A (MTA)  --- SMTP (port 25) ---→  Mail Server B (MTA)
     |                                              |
     └── MDA (doručení do schránky) ←───────────────┘

MUA = Mail User Agent (klient: Thunderbird, Outlook, Gmail web)
MTA = Mail Transfer Agent (server: Postfix, Exim, Sendmail)
MDA = Mail Delivery Agent (doručení do mailboxu: Dovecot)

SMTP (Simple Mail Transfer Protocol)

Funkce

  • Odesílání e-mailů od klienta na server
  • Přenos e-mailů mezi servery (relay)
  • Textový protokol (příkaz → odpověď s kódem)

SMTP porty

PortPoužitíZabezpečení

IMAP (Internet Message Access Protocol)

Funkce

  • E-maily zůstávají na serveru
  • Klient synchronizuje stav se serverem (přečteno, označeno, smazáno)
  • Podpora složek (Inbox, Sent, Drafts, Trash…)
  • Více klientů přistupuje ke stejné schránce
  • IDLE příkaz → push notifikace o nových e-mailech

Pro maturitu: SMTP = odesílání (port 587/465). IMAP = příjem + synchronizace se serverem (port 993). POP3 = stažení a smazání ze serveru (port 995). MIME = přílohy. SPF/DKIM/DMARC = ochrana proti spoofingu.
