🧬 ČÁST 2 — Generika

Generika

  • Parametrizované typy — kód funguje s různými typy
  • Zajišťují typovou bezpečnost při kompilaci (bez castování)
  • C#: List<T>, Dictionary<K,V>
  • Java: List<T> (type erasure)
  • Python: list[int] (type hints, nevalidováno za běhu)
C# — generická třída — Seznam<T> funguje pro libovolný typ; kompilátor ověří typy při kompilaci.
class Seznam<T> {
    private T[] items = new T[10];
    private int count = 0;
    public void Pridej(T item) { items[count++] = item; }
    public T Get(int i) => items[i];
}
var cisla = new Seznam<int>();
var texty = new Seznam<string>();
Java — generická třída — Box<T>; Java používá type erasure — za běhu je generický typ smazán.
class Box<T> {
    private T value;
    public void set(T v) { value = v; }
    public T get() { return value; }
}
Box<Integer> box = new Box<>();
box.set(42);

Struct (struktury) — C#

  • Value type — uložen na stacku, kopíruje se při přiřazení (ne reference)
  • Vhodný pro malá data (souřadnice, barva, rozměr)
  • Nemůže dědit z jiné struct/class (ale může implementovat interface)
  • Může mít konstruktor, metody a přetížené operátory
struct Bod { public int X, Y; }
var b1 = new Bod { X = 1, Y = 2 };

Ukázkový kód — C#

[Flags] Enum — bitové příznaky, umožňuje kombinovat více hodnot pomocí | operátoru.
[Flags]
enum Opravneni {
    Zadne   = 0,
    Cteni   = 1,
    Zapis   = 2,
    Mazani  = 4,
    Admin   = Cteni | Zapis | Mazani  // kombinace = 7
}

var prava = Opravneni.Cteni | Opravneni.Zapis;
Console.WriteLine($"Prava: {prava}");          // Cteni, Zapis
Console.WriteLine($"Ma zapis? {prava.HasFlag(Opravneni.Zapis)}"); // True
Struct s přetížením operátorů — hodnotový typ Bod, operátor + sčítá souřadnice, == porovnává složky.
struct Bod {
    public double X { get; }
    public double Y { get; }

    public Bod(double x, double y) { X = x; Y = y; }

    public static Bod operator +(Bod a, Bod b) => new Bod(a.X + b.X, a.Y + b.Y);
    public static bool operator ==(Bod a, Bod b) => a.X == b.X && a.Y == b.Y;
    public static bool operator !=(Bod a, Bod b) => !(a == b);
    public override string ToString() => $"({X}, {Y})";
}

var p1 = new Bod(1, 2);
var p2 = new Bod(3, 4);
Console.WriteLine($"{p1} + {p2} = {p1 + p2}"); // (4, 6)
Generický Stack<T> — typově bezpečný zásobník. Constraint where T : IComparable<T> vyžaduje porovnatelnost prvků pro metodu Min().
class MujStack<T> {
    private T[] prvky;
    private int vrchol = -1;

    public MujStack(int kapacita = 10) {
        prvky = new T[kapacita];
    }

    public void Push(T hodnota) {
        prvky[++vrchol] = hodnota;
    }

    public T Pop() {
        return prvky[vrchol--];
    }

    public T Peek() => prvky[vrchol];
    public bool JePrazdny() => vrchol == -1;
}

// Genericke omezeni
static T NajdiMin<T>(T[] pole) where T : IComparable<T> {
    T min = pole[0];
    foreach (var prvek in pole) {
        if (prvek.CompareTo(min) < 0)
            min = prvek;
    }
    return min;
}

var stack = new MujStack<int>();
stack.Push(10);
stack.Push(20);
Console.WriteLine($"Pop: {stack.Pop()}");  // 20

int[] cisla = { 5, 3, 8, 1, 9 };
Console.WriteLine($"Min: {NajdiMin(cisla)}"); // 1
Nullable hodnotové typy — int? umožňuje null pro hodnotové typy. Null-coalescing (??) a null-conditional (?.) operátory.
int? moznaNull = null;
int hodnota = moznaNull ?? 42;  // null-coalescing
Console.WriteLine($"Hodnota: {hodnota}");  // 42

string? text = null;
int? delka = text?.Length;  // null-conditional
Console.WriteLine($"Delka: {delka ?? 0}"); // 0
