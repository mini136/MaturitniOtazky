Ukázkový kód — Java (dědičnost, overriding, interface)

Abstraktní třída Tvar — definuje společný kontrakt (barva, abstraktní obsah() a obvod()). Potomci Kruh a Obdelnik implementují výpočty.
abstract class Tvar {
    protected String barva;

    public Tvar(String barva) {
        this.barva = barva;
    }

    abstract double obsah();
    abstract double obvod();

    public void info() {
        System.out.printf("  %s [%s]: obsah=%.2f, obvod=%.2f%n",
            getClass().getSimpleName(), barva, obsah(), obvod());
    }
}

class Kruh extends Tvar {
    private double polomer;

    public Kruh(double polomer, String barva) {
        super(barva);
        this.polomer = polomer;
    }

    @Override
    double obsah() { return Math.PI * polomer * polomer; }

    @Override
    double obvod() { return 2 * Math.PI * polomer; }
}

class Obdelnik extends Tvar {
    protected double sirka, vyska;

    public Obdelnik(double sirka, double vyska, String barva) {
        super(barva);
        this.sirka = sirka;
        this.vyska = vyska;
    }

    @Override
    double obsah() { return sirka * vyska; }

    @Override
    double obvod() { return 2 * (sirka + vyska); }
}
Víceúrovňová dědičnost — Ctverec dědí z Obdelnik (šířka = výška). Přepisuje toString().
class Ctverec extends Obdelnik {
    public Ctverec(double strana, String barva) {
        super(strana, strana, barva);
    }

    @Override
    public String toString() {
        return String.format("Ctverec(strana=%.1f, %s)", sirka, barva);
    }
}

// Polymorfismus - pole rodičovského typu
Tvar[] tvary = {
    new Kruh(5, "cervena"),
    new Obdelnik(4, 6, "modra"),
    new Ctverec(3, "zelena")
};
for (Tvar t : tvary) {
    t.info();  // volá správnou implementaci díky overriding
}
Interface — Kreslitelny a Zmensitelny. Třída může implementovat více rozhraní (na rozdíl od jedné rodičovské třídy).
interface Kreslitelny {
    void nakresli();
}

interface Zmensitelny {
    void zmensNa(double procento);
}

// Trida implementujici vice interfacu
class KreslitelnyKruh extends Kruh implements Kreslitelny, Zmensitelny {
    private double polomer;

    public KreslitelnyKruh(double polomer, String barva) {
        super(polomer, barva);
        this.polomer = polomer;
    }

    @Override
    public void nakresli() {
        System.out.println("  Kreslim kruh s polomerem " + polomer);
    }

    @Override
    public void zmensNa(double procento) {
        polomer *= (procento / 100.0);
        System.out.println("  Zmenseno na " + procento + "%, novy polomer=" + polomer);
    }
}
Overloading (přetěžování) — stejný název metody secti, ale různé typy/počty parametrů. Rozhoduje se při kompilaci.
class Kalkulacka {
    static int secti(int a, int b) { return a + b; }
    static double secti(double a, double b) { return a + b; }
    static int secti(int a, int b, int c) { return a + b + c; }
    static String secti(String a, String b) { return a + b; }
}

System.out.println("int:    " + Kalkulacka.secti(3, 5));          // 8
System.out.println("double: " + Kalkulacka.secti(3.5, 2.1));     // 5.6
System.out.println("3 args: " + Kalkulacka.secti(1, 2, 3));      // 6
System.out.println("String: " + Kalkulacka.secti("Ahoj ", "!")); // Ahoj !
instanceof — kontrola typu za běhu. Umožňuje bezpečné přetypování (downcasting).
Tvar tvar = new Kruh(5, "modra");
if (tvar instanceof Kruh) {
    Kruh k = (Kruh) tvar;
    System.out.println("Je to kruh s polomerem");
}

// Java 16+ pattern matching
if (tvar instanceof Kruh k) {
    System.out.println("Pattern matching: " + k.obsah());
}
