03 – Rekurze, brute force, heuristiky

Rekurze

Princip

  • Funkce volá sama sebe (přímo nebo nepřímo)
  • Bázový případ — podmínka ukončení (zabraňuje nekonečné rekurzi)
  • Rekurzivní krok — zmenšení problému a volání sebe sama
  • Každé volání vytváří nový rámec na zásobníku (stack frame)

Typy rekurze

TypPopis
PřímáFunkce volá přímo samu sebe
NepřímáA volá B, B volá A
Tail rekurzeRekurzivní volání je poslední operací — možná optimalizace
StromováVíce rekurzivních volání (Fibonacci)

Brute Force

  • Systematické prohledání všech možných řešení
  • Vždy najde optimální řešení (pokud existuje)
  • Složitost typicky exponenciální: O(2ⁿ), O(n!)
  • Použitelný pouze pro malé vstupy

Heuristiky

  • Greedy — v každém kroku vybere lokálně nejlepší volbu (negarantuje globální optimum)
  • Hill Climbing — vylepšuje řešení lokálními změnami, může uvíznout v lokálním maximu
  • Monte Carlo — náhodné vzorkování pro odhad řešení
  • Genetické algoritmy — evoluční přístup (populace, selekce, křížení, mutace)

Ukázkový kód — Python

Faktoriál — klasický příklad přímé rekurze. Bázový případ: n ≤ 1.
def faktorial(n):
    if n <= 1:
        return 1
    return n * faktorial(n - 1)

print(f"5! = {faktorial(5)}")  # 120
Fibonacci s memoizací — naivní rekurze má O(2ⁿ), s cache (memoizací) dosáhneme O(n).
def fibonacci(n, cache={}):
    """S memoizaci - O(n)"""
    if n in cache:
        return cache[n]
    if n <= 1:
        return n
    cache[n] = fibonacci(n-1, cache) + fibonacci(n-2, cache)
    return cache[n]

print(f"fib(10) = {fibonacci(10)}")  # 55
Hanojské věže — rekurzivní přesun n disků mezi třemi kolíky. Důkaz O(2ⁿ - 1) tahů.
def hanoi(n, zdroj='A', cil='C', pomocny='B'):
    """Hanojske veze - O(2^n - 1)"""
    if n == 1:
        print(f"  Presun disk 1: {zdroj} -> {cil}")
        return
    hanoi(n-1, zdroj, pomocny, cil)
    print(f"  Presun disk {n}: {zdroj} -> {cil}")
    hanoi(n-1, pomocny, cil, zdroj)

hanoi(3)
Problém batohu — brute force přístup vyzkouší všech 2ⁿ kombinací a vybere nejhodnotnější sadu, co se vejde do kapacity.
from itertools import combinations

def batoh_bruteforce(predmety, kapacita):
    """Brute force batoh - O(2^n)"""
    nejlepsi_hodnota = 0
    nejlepsi_sada = []

    for r in range(len(predmety) + 1):
        for sada in combinations(predmety, r):
            vaha = sum(p['vaha'] for p in sada)
            hodnota = sum(p['hodnota'] for p in sada)
            if vaha <= kapacita and hodnota > nejlepsi_hodnota:
                nejlepsi_hodnota = hodnota
                nejlepsi_sada = list(sada)

    return nejlepsi_sada, nejlepsi_hodnota

predmety = [
    {'nazev': 'Notebook', 'vaha': 3, 'hodnota': 10},
    {'nazev': 'Tablet',   'vaha': 1, 'hodnota': 6},
    {'nazev': 'Kamera',   'vaha': 2, 'hodnota': 7},
    {'nazev': 'Kniha',    'vaha': 1, 'hodnota': 3}
]
sada, hodnota = batoh_bruteforce(predmety, 4)
print(f"Nejlepsi sada (hodnota={hodnota}): {[p['nazev'] for p in sada]}")
Greedy heuristika — seřadí předměty podle poměru hodnota/váha a vybírá postupně. Nezaručuje optimální řešení, ale je rychlá O(n log n).
def batoh_greedy(predmety, kapacita):
    """Greedy - razeni podle pomeru hodnota/vaha"""
    serazene = sorted(predmety, key=lambda p: p['hodnota']/p['vaha'], reverse=True)
    sada = []
    zbyvajici = kapacita
    for p in serazene:
        if p['vaha'] <= zbyvajici:
            sada.append(p)
            zbyvajici -= p['vaha']
    return sada, sum(p['hodnota'] for p in sada)

sada_g, hodnota_g = batoh_greedy(predmety, 4)
print(f"Greedy (hodnota={hodnota_g}): {[p['nazev'] for p in sada_g]}")
Genetický algoritmus — evoluční přístup: vytvoří populaci náhodných řešení, provádí selekci, křížení a mutaci přes generace.
import random

def geneticky_algoritmus(predmety, kapacita, velikost_populace=20, generace=50):
    n = len(predmety)

    def fitness(jedinec):
        vaha = sum(p['vaha'] for p, bit in zip(predmety, jedinec) if bit)
        hodnota = sum(p['hodnota'] for p, bit in zip(predmety, jedinec) if bit)
        return hodnota if vaha <= kapacita else 0

    # Inicializace populace
    populace = [[random.randint(0,1) for _ in range(n)]
                for _ in range(velikost_populace)]

    for gen in range(generace):
        populace.sort(key=fitness, reverse=True)
        nova = populace[:4]  # elitismus - nejlepsi preziji
        while len(nova) < velikost_populace:
            rodic1, rodic2 = random.choices(populace[:10], k=2)
            bod = random.randint(1, n-1)
            dite = rodic1[:bod] + rodic2[bod:]
            # Mutace
            if random.random() < 0.1:
                i = random.randint(0, n-1)
                dite[i] = 1 - dite[i]
            nova.append(dite)
        populace = nova

    nejlepsi = max(populace, key=fitness)
    return [p for p, bit in zip(predmety, nejlepsi) if bit], fitness(nejlepsi)

sada_ga, hod_ga = geneticky_algoritmus(predmety, 4)
print(f"Geneticky (hodnota={hod_ga}): {[p['nazev'] for p in sada_ga]}")
