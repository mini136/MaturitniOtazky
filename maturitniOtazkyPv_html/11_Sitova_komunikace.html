🔌 ČÁST 3 — Berkeley Socket API

Berkeley Socket API

Životní cyklus soketu

  1. socket() — vytvoření soketu
  2. bind() — přiřazení adresy a portu (server)
  3. listen() — naslouchání (server)
  4. accept() — přijetí spojení (server)
  5. connect() — připojení k serveru (klient)
  6. send()/recv() — výměna dat
  7. close() — uzavření

Ukázkový kód — Python (TCP server/klient, UDP, HTTP)

TCP Echo server — naslouchá na portu, každé připojení obsluhuje v samostatném vlákně. Přijatá data odešle zpět velkými písmeny (echo).
import socket
import threading

def tcp_server():
    """TCP echo server s vlakny"""
    server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    server.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
    server.bind(('localhost', 65432))
    server.listen(5)
    print("Server posloucha na localhost:65432")

    def obsluha_klienta(conn, addr):
        print(f"Pripojeni od {addr}")
        with conn:
            while True:
                data = conn.recv(1024)
                if not data:
                    break
                odpoved = data.decode().upper()
                conn.sendall(odpoved.encode())
        print(f"Odpojeni {addr}")

    try:
        while True:
            conn, addr = server.accept()
            vlakno = threading.Thread(target=obsluha_klienta, args=(conn, addr))
            vlakno.daemon = True
            vlakno.start()
    finally:
        server.close()
TCP klient — připojí se k serveru, odešle zprávu, přijme odpověď. Používá with pro bezpečné uzavření soketu.
def tcp_klient():
    """TCP klient"""
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
        s.connect(('localhost', 65432))

        zpravy = ["Ahoj servere!", "Jak se mas?", "Nashledanou"]
        for zprava in zpravy:
            s.sendall(zprava.encode())
            data = s.recv(1024)
            print(f"Odeslano: '{zprava}' -> Prijato: '{data.decode()}'")
UDP server a klient — bezspojový protokol. recvfrom() vrací data i adresu odesílatele, sendto() odesílá na konkrétní adresu.
def udp_server():
    """UDP datagram server"""
    server = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
    server.bind(('localhost', 65433))
    print("UDP server na portu 65433")

    while True:
        data, addr = server.recvfrom(1024)   # bez accept!
        print(f"UDP od {addr}: {data.decode()}")
        server.sendto(data.upper(), addr)     # odpoved na adresu odesilatele

def udp_klient():
    """UDP klient"""
    with socket.socket(socket.AF_INET, socket.SOCK_DGRAM) as s:
        s.sendto(b"Hello UDP!", ('localhost', 65433))
        data, addr = s.recvfrom(1024)
        print(f"UDP odpoved: {data.decode()}")
Raw HTTP GET — nízkoúrovňový HTTP požadavek přes TCP soket. Ručně sestavíme HTTP hlavičku a zpracujeme odpověď.
def http_get(host, cesta="/"):
    """HTTP GET pozadavek pomoci raw soketu"""
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
        s.connect((host, 80))

        pozadavek = f"GET {cesta} HTTP/1.1\r\nHost: {host}\r\nConnection: close\r\n\r\n"
        s.sendall(pozadavek.encode())

        odpoved = b""
        while True:
            cast = s.recv(4096)
            if not cast:
                break
            odpoved += cast

        text = odpoved.decode('utf-8', errors='ignore')
        hlavicka, _, telo = text.partition('\r\n\r\n')
        print(f"Status: {hlavicka.split(chr(10))[0]}")
        print(f"Telo (prvnich 200 znaku): {telo[:200]}")
