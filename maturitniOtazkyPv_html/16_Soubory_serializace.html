📁 ČÁST 1 — Práce se soubory

Práce se soubory

MódPopisPříklad
'r'Čtení (soubor musí existovat)open('f.txt', 'r')
'w'Zápis (přepis souboru, vytvoří nový)open('f.txt', 'w')
'a'Přidání na konec (append)open('f.txt', 'a')

Best practice: Vždy používát with open(...) as f: — automaticky zavře soubor i při výjimce. Textový mód převádí konce řádků dle OS; binární mód ne.

Kódování: Textové soubory v Python 3 otevřít vždy s encoding='utf-8'. Bez toho závisí na OS (Windows používá cp1250). Příklad: open('f.txt', 'r', encoding='utf-8')

Ukázkový kód — Python

Textové soubory — zápis ('w') a čtení ('r') pomocí with (automatické zavření). Mode 'a' pro přidání na konec.
import os

# Zapis do souboru
with open("test.txt", "w", encoding="utf-8") as f:
    f.write("Prvni radek\n")
    f.write("Druhy radek\n")
    f.writelines(["Treti\n", "Ctvrty\n"])

# Cteni celeho souboru
with open("test.txt", "r", encoding="utf-8") as f:
    obsah = f.read()
    print(obsah)

# Cteni po radcich (setrne k pameti)
with open("test.txt", "r", encoding="utf-8") as f:
    for radek in f:
        print(f"  Radek: {radek.strip()}")

os.remove("test.txt")
JSON — serializace slovníku do JSON stringu a zpět. json.dumps() pro řetězec, json.dump() pro soubor. Parametr ensure_ascii=False pro české znaky.
import json

data = {
    "jmeno": "Alice",
    "vek": 20,
    "predmety": ["PV", "DS", "WA"],
    "aktivni": True
}

# Do souboru
with open("student.json", "w", encoding="utf-8") as f:
    json.dump(data, f, indent=2, ensure_ascii=False)

# Ze souboru
with open("student.json", "r", encoding="utf-8") as f:
    nacteno = json.load(f)
    print(f"  JSON: {nacteno}")

# Do/z retezce
json_str = json.dumps(data, ensure_ascii=False)
zpet = json.loads(json_str)

os.remove("student.json")
CSV — čtení a zápis tabulkových dat. DictWriter zapisuje slovníky s hlavičkou, DictReader čte řádky jako slovníky.
import csv

# Zapis CSV
studenti = [
    {"jmeno": "Alice", "vek": 20, "prumer": 1.5},
    {"jmeno": "Bob", "vek": 22, "prumer": 2.1}
]

with open("studenti.csv", "w", newline="", encoding="utf-8") as f:
    writer = csv.DictWriter(f, fieldnames=["jmeno", "vek", "prumer"])
    writer.writeheader()
    writer.writerows(studenti)

# Cteni CSV
with open("studenti.csv", "r", encoding="utf-8") as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(f"  {row['jmeno']}: vek={row['vek']}, prumer={row['prumer']}")

os.remove("studenti.csv")
XML — vytvoření XML stromu pomocí ElementTree, zápis do souboru a zpětné parsování s procházením elementů.
import xml.etree.ElementTree as ET

# Vytvoreni XML
root = ET.Element("studenti")
for jmeno, vek in [("Alice", "20"), ("Bob", "22")]:
    student = ET.SubElement(root, "student")
    ET.SubElement(student, "jmeno").text = jmeno
    ET.SubElement(student, "vek").text = vek

tree = ET.ElementTree(root)
tree.write("studenti.xml", encoding="utf-8", xml_declaration=True)

# Parsovani XML
tree = ET.parse("studenti.xml")
for student in tree.findall("student"):
    jmeno = student.find("jmeno").text
    vek = student.find("vek").text
    print(f"  XML: {jmeno}, vek: {vek}")

os.remove("studenti.xml")
Pickle — binární serializace Python objektů. POZOR: nebezpečné pro nedůvěryhodná data! A struct — nízkoúrovňové binární pakování (big-endian integer + float).
import pickle
import struct

# Pickle - Python objekty (NEBEZPECNE pro neduveryhodna data!)
data = {"klice": [1, 2, 3], "vnorene": {"a": True}}
with open("data.pkl", "wb") as f:
    pickle.dump(data, f)

with open("data.pkl", "rb") as f:
    nacteno = pickle.load(f)
    print(f"  Pickle: {nacteno}")
os.remove("data.pkl")

# Struct - binarni data
# >if = big-endian int + float
packed = struct.pack(">if", 42, 3.14)
print(f"  Packed: {packed.hex()}")

unpacked = struct.unpack(">if", packed)
print(f"  Unpacked: {unpacked}")
