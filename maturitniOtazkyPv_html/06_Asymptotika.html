Pravidla

  • Dominantní člen: O(3n² + 5n + 7) = O(n²)
  • Konstanty se ignorují: O(5n) = O(n)
  • Výběr nejhoršího: Sekvenční bloky — O(n) + O(n²) = O(n²)
  • Vnořené cykly: O(n) × O(n) = O(n²)

Amortizovaná složitost

Průměrná složitost přes sérii operací. Například: list.append() v Pythonu je amortizovaně O(1), i když občas dochází k realokaci O(n).

Ukázkový kód — Python (měření složitostí)

O(1) — konstantní: Přístup k prvku pole indexem nezávisí na velikosti pole.
import time

def o_1(pole):
    """O(1) - pristup k prvku indexem"""
    return pole[0]

pole = list(range(1000000))
start = time.perf_counter()
o_1(pole)
cas = time.perf_counter() - start
print(f"O(1): {cas:.8f}s")
O(log n) — logaritmická: Binární vyhledávání půlí interval v každém kroku.
def o_log_n(pole, hledany):
    """O(log n) - binarni vyhledavani"""
    levy, pravy = 0, len(pole) - 1
    while levy <= pravy:
        stred = (levy + pravy) // 2
        if pole[stred] == hledany:
            return stred
        elif pole[stred] < hledany:
            levy = stred + 1
        else:
            pravy = stred - 1
    return -1

index = o_log_n(pole, 999999)
print(f"O(log n): nalezeno na indexu {index}")
O(n) — lineární: Průchod celým polem, O(n²) — kvadratická: Dva vnořené cykly.
def o_n(pole):
    """O(n) - linearni pruchod"""
    soucet = 0
    for prvek in pole:
        soucet += prvek
    return soucet

def o_n2(pole):
    """O(n²) - dva vnorene cykly"""
    pocet = 0
    for i in range(len(pole)):
        for j in range(len(pole)):
            pocet += 1
    return pocet

# Mereni s ruznymi velikostmi
for n in [100, 1000, 5000]:
    data = list(range(n))
    start = time.perf_counter()
    o_n2(data)
    cas = time.perf_counter() - start
    print(f"O(n²) pro n={n}: {cas:.4f}s")
O(2ⁿ) — exponenciální: Naivní rekurzivní Fibonacci — každé volání se větví na dvě podvolání.
def o_2n(n):
    """O(2^n) - rekurzivni fibonacci"""
    if n <= 1:
        return n
    return o_2n(n - 1) + o_2n(n - 2)

for n in [10, 20, 30]:
    start = time.perf_counter()
    vysledek = o_2n(n)
    cas = time.perf_counter() - start
    print(f"O(2^n) pro n={n}: {cas:.4f}s (vysledek={vysledek})")
